## âœ¨ Features

- **Smart Scheduling**: Uses Google OR-Tools constraint programming to find optimal schedules
- **Role-Based Assignments**: Supports ATL (managers), FullTime, and PartTime roles
- **Availability-Aware**: Respects employee availability and maximum weekly hours
- **Target Hour Optimization**: Tries to schedule employees close to their preferred hours
- **Web Interface**: Modern, responsive schedule viewer with dark mode
- **Export Options**: JSON for web, CSV for spreadsheets

## ğŸ“Š How It Works

The scheduler uses **Constraint Programming** to find assignments that:

1. **Fill all shifts** with exactly one employee each
2. **Match roles** (ATL shifts to ATL employees, etc.)
3. **Respect availability** (only schedule during available hours)
4. **Limit daily shifts** (max 1 shift per day per employee)
5. **Cap weekly hours** (never exceed max_weekly_hours)
6. **Maximize satisfaction** (get everyone close to their target hours)
